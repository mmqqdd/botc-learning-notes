## 一、剧本机制总览

### 跟暗流涌动的本质差异

| 维度 | 暗流涌动（入门） | 横行霸道 |
|---|---|---|
| 主题 | 信息与假信息 | **明牌邪方**：阵营随时可能转变 |
| 镇民 | 13 | 13（**完全不同**） |
| 外来者 | 4 | **5**（多 1 个） |
| 爪牙 | 4 | **5**（多 1 个） |
| 恶魔 | 1（小恶魔） | **2**（痢蛭 + 小恶魔） |
| 传奇 | 无 | 1（哨兵：±1 外来者） |

### 善方的胜利路径

经典胜利：处决恶魔
- 但横行霸道**有 2 个恶魔**，且**痢蛭可能不会死**（要先杀宿主）
- **核心信息系**：占卜师、博学者、将军、杂耍艺人、渔夫
- **关键防御**：僧侣保护强神，水手免死，茶艺师保护邻居

### 邪方的胜利路径

经典胜利：杀人杀到只剩 2 个活人
- 但有**5 个"特殊胜负开关"角色**可以**绕过常规**直接赢/输——这是横行霸道的精髓
- 详见 [② 必背关键互动](#二-必背关键互动-胜负开关)

## 二、必背关键互动 · 胜负开关

> **下面这几条决定输赢，必须背下来。**很多新手输都不是输在杀人战术，是输在不懂这些机制。

### 1. 哥布林公开声明 → 邪方直接赢

**条件**：哥布林**被提名后公开声明自己是哥布林**，且**在那个白天被处决**。

**善方怎么防**：
- 听到有人说"我是哥布林"时**立刻撤回提名**——千万别真投他
- 同理，任何人**自爆"我是邪方"时都别冲动**——可能是哥布林骗局
- 哥布林靠"挑衅"工作，他要主动求死。**冷处理**他就废了

### 2. 落难少女被爪牙猜中 → 善方直接输

**条件**：任意爪牙**公开猜测某人是落难少女**且猜对（一局只能猜一次）。

**善方怎么防**：
- **落难少女永远别跳身份**——一跳就送
- 善方主动**装假落难少女**：故意几个人表现得像落难少女，让爪牙赌错
- 落难少女被怀疑时主动**跳一个不矛盾的强神**（如博学者）让爪牙不敢猜你

### 3. 恐惧之灵提名 + 处决目标 → 目标阵营输

**条件**：恐惧之灵**亲自提名**他选中的玩家，且该提名导致目标**被处决**。

**信号**：每当恐惧之灵**首次选择或更换目标**时，**所有人会被告知**（但不知道是谁）——这是一个公开警示。

**善方怎么防**：
- 听到"恐惧之灵选择了新目标"广播时，**不要轻易让任何人提名处决**——尤其那个总在提名别人的人就是恐惧之灵的高危候选
- 善方应该**频繁提名**降低恐惧之灵单杀的概率（恐惧之灵提名 ≠ 自动赢，还要被处决）

### 4. 政客是元凶 → 转阵营获胜

**条件**：政客是**对己方阵营落败负最大责任**的人，转邪并获胜（即使已死）。

**善方怎么防**：
- 处决疑似政客前**先想想**：他疯狂带节奏、误导大家投错人——可能不是邪方，是政客想赢
- 政客**没有明示标记**，只能靠行为推断

### 5. 异端分子在场 → 胜负对调

**条件**：异端分子**在场**（不论生死），最终胜负**反过来**。

**善方怎么防**：
- **不能保留异端分子到最后**——保留他你赢了等于输
- 但邪方也想保异端分子——他不死，你赢了就变邪方赢
- 如果场上**确认存在异端分子**，善方目标变成：**让异端分子被处决**，然后正常赢

### 6. 痢蛭恶魔的"宿主"机制

**条件**：首夜痢蛭选一个**宿主**（中毒），**只有宿主死亡，痢蛭才会死**。处决痢蛭只会被告知"他还活着"。

**善方怎么解谜**：
- 第一夜后，每个善方都可能是潜在宿主
- **谁有信息异常 = 可能是宿主**（中毒导致信息错乱）
- 处决痢蛭如果"未死"，**下一步就是找出宿主**——盘谁的信息一直不对

### 7. 精神病患者的猜拳免死

**条件**：精神病患者每天**公开选一人死**。被处决时，他和提名者**猜拳**，输的死。

**善方怎么防**：
- 善方提名精神病患者**有 50% 概率白送**（他赢了你死）
- 多人轮流提名：如果你输了猜拳就死，**只能祈祷下一个人赢**
- 实战：**优先用其他爪牙清完场再处决精神病患者**

## 三、上桌前 5 分钟 checklist

> 上桌前过一遍这页。这是这剧本的"操作系统级 facts"。

### 配置基础

```
□ 13 镇民 + 5 外来者 + 5 爪牙 + 2 恶魔（剧本池）
□ 实际局可能因哨兵 ±1 外来者，因莽夫/政客转阵营
□ 2 个恶魔：痢蛭（不死）+ 小恶魔（标准）
□ 罂粟种植者活着时，爪牙恶魔互相不认识
```

### 胜负陷阱（必检查）

```
□ 哥布林：他公开自爆别冲动投他
□ 落难少女：自己是落难少女绝不跳身份
□ 恐惧之灵：听到"换目标"广播就要警惕谁在提名
□ 政客：疯狂带节奏的人可能是政客不是邪方
□ 异端分子：他在场胜负对调，必须处决他
```

### 信息可信度

```
□ 信息系镇民可能被投毒者毒 → 信息错乱别冤枉好人
□ 痢蛭宿主中毒 → 持续给错信息
□ 莽夫被恶魔/爪牙选中 → 他转邪了，原本的善方信息废
□ 中毒/醉酒玩家：信息全错
```

### 跳身份原则

```
□ 强信息源（占卜师/博学者/将军/杂耍艺人）：D1-D2 跳
□ 防御系（僧侣/茶艺师）：能不跳就不跳
□ 一局一次（渔夫/巡山人）：能力没用完别跳
□ 邪方该装：博学者/占卜师/调查员（信息系，验证难）
□ 邪方别装：水手（容易被验证不死）、僧侣（保护对象会暴露）
```

## 四、拿到善方怎么玩

### 信息系镇民（5 个核心）

| 角色 | 关键策略 |
|---|---|
| **占卜师** | D1 选最可疑的两人对验；得到"有恶魔"信号别立刻锁人——可能是红鲱鱼 |
| **博学者** | D1 私下问说书人；问"如果错了能反推真相"的二元问题 |
| **杂耍艺人** | **D1 必须公开猜 5 次**；优先猜跳了身份的人（对了证善方，错了锁邪方） |
| **将军** | 风向标，**先观察 2-3 晚再跳**；连续指向邪恶 = 急 |
| **渔夫** | **保险卡**，关键决策时找说书人要建议——别浪费 |

### 状态/设置触发型镇民

| 角色 | 关键策略 |
|---|---|
| **炼金术士** | 你拥有不在场爪牙能力，**用善方思路使用**；爪牙以为你是队友会暴露 |
| **舞蛇人** | 首夜选最可疑的人；选中恶魔 = 交换+你中毒，但替善方除掉一个恶魔 |
| **罂粟种植者** | **死亡是你最大贡献**——你活着邪方就被致盲。**藏好让他们找不到你** |
| **巡山人** | 一局一次，**强烈线索锁定落难少女**才用，否则浪费 |
| **食人族** | 别先跳，看一两轮处决再亮身份；处决邪方 = 你中毒注意 |

### 防御系镇民

| 角色 | 关键策略 |
|---|---|
| **僧侣** | 优先保护**已跳的信息系强神**（占卜师/博学者/杂耍艺人） |
| **水手** | **不会死**，可以非常激进地跳身份；副作用是有人醉 |
| **茶艺师** | **别跳**让邪方不知道你存在；尽量挨着两个确认善方 |

## 五、拿到邪方怎么玩 ⭐（重点章节）

> 这是你最关心的部分。所有邪方角色都有一个共同任务：**装成善良玩家**，同时**完成自己的特殊任务**。

### 5.1 邪方第一夜流程

**首夜你会被叫醒**：
- 知道队友是谁（**除非罂粟种植者在场，那就不知道**）
- 知道恶魔得到的 **3 个"伪角色"**——这些角色**不在本局**，可以用来跳身份

**首夜立刻盘**：
1. 队友有谁（如果不知道，说明罂粟种植者在场）
2. 自己手上的能力是什么
3. 准备 D1 跳什么身份（用伪角色之一）

### 5.2 邪方该跳什么身份 ⭐

**核心原则**：跳**信息系**最稳，跳**容易被验证**的角色会死。

| 推荐跳的身份 | 为什么 |
|---|---|
| **博学者** ⭐⭐⭐ | 你说"私下问说书人得到的"，没人能反验。**最稳的假身份** |
| **占卜师** ⭐⭐⭐ | 有"红鲱鱼"机制——验错可以解释为干扰项 |
| **将军** ⭐⭐ | 抽象信号（"邪恶/善良/均势"），随便编 |
| **杂耍艺人** ⭐⭐ | D1 公开猜测，可以精确选错让自己看起来"普通善方" |
| **渔夫** ⭐ | 一局一次能力，能不暴露 |
| **食人族** ⭐ | 没有主动行动，被动型，最难验证 |

**别跳的身份**：
- **水手**：会有人发现你被刀了竟然没死（你是真的会死）
- **僧侣**：你"保护"的人晚上被刀了 = 你身份废
- **茶艺师**：你邻居死了 = 你身份废
- **罂粟种植者**：你跳出来后队友互认马上启动，逻辑漏洞大
- **舞蛇人**：你说选中了恶魔但身份没变 = 显然在编

### 5.3 邪方该刀谁

按威胁度优先级排序：

```
1. 罂粟种植者（消除邪方致盲，最高优先级）
2. 跳了身份的信息系强神（博学者、占卜师、杂耍艺人、将军）
3. 还没跳但行为像信息系的人
4. 防御系（僧侣 → 保护别人）

刀别人无意义的角色：
- 水手（刀不死）
- 茶艺师（要刀她邻居，茶艺师本人无用）
- 已开过能力的猎手（不在本剧本，但同理）
- 已死的玩家
```

### 5.4 各爪牙的核心打法

#### 投毒者 Poisoner

**核心任务**：**破坏信息**。横行霸道信息源多，投毒者价值巨大。

**毒谁**：
- D1 毒最强的信息系角色（如博学者）→ 他得到两条错信息
- D2 之后毒**跳出来被信任的人**——让大家相信他给出的错信息
- **千万别毒已死的人**——浪费

**伪信号**：
- 你毒了博学者 → 他公布的信息会自相矛盾
- 善方会怀疑博学者，**主动跟善方一起怀疑**（"博学者怎么前后矛盾"）

#### 恐惧之灵 Fearmonger

**核心任务**：**用单杀机制赢游戏**。

**操作流程**：
1. 首夜选一个目标（被告知"恐惧之灵选了人"——但你只在 D1 选一次保持安静）
2. 装一个被信任的善方身份（博学者最好）
3. **某天提名你的目标**——但要**等他真的会被处决**那一天再提名
4. 善方处决你的目标 → 直接赢

**避坑**：
- 别 D1 就着急提名——**没建立信任之前提名很容易被怀疑**
- **目标选谁？最难被善方"反指认"的人**——例如某个不太活跃的玩家
- **不要频繁换目标**——每换一次会广播警示

#### 哥布林 Goblin

**核心任务**：**自爆+被处决=赢**。

**两种打法**：
- **激进派**：演得很可疑（不参与讨论、说些奇怪的话），让善方提名你，被提名后立即自爆"我是哥布林！"——如果善方头一热投了你，赢
- **保守派**：装跳强神（如博学者），被善方发现身份后**主动暴露**自爆

**避坑**：
- 提名后才能自爆——**别在提名前主动说**
- **大多数老玩家会冷处理"我是哥布林"**——他们撤回提名，你就废
- 配合其他爪牙：让某个"假善方"提名你，营造氛围

#### 魔鬼代言人 Devil's Advocate

**核心任务**：**保命强爪牙/恶魔**。

**保谁**：
1. 优先保**恶魔**（特别是身份要暴露的小恶魔）
2. 其次保**强爪牙**（恐惧之灵、精神病患者）
3. 自己保自己（如果你被怀疑）

**操作**：
- 每晚必须换不同目标——**不能连续保同一个人**
- 被保护的人即使被处决也不死——**说书人只说"被处决"不说原因**，可以伪装"哎呀我居然没事"装强神

#### 精神病患者 Psychopath

**核心任务**：**每天公开杀+猜拳赖活**。

**杀谁**：
1. **最强的信息系**（D1 杀博学者一个就废一半场）
2. **最确认的善方**——他们越确认，你杀越亏（善方）
3. 别杀**红鲱鱼**目标（看着像但其实没价值）

**猜拳免死**：
- 你被提名 → 提名者跟你猜拳 → 你赢=他死，你输=你死
- **多人轮流提名你**：你只要每次猜赢就活——50% 概率赖活
- 善方知道这点会**多人陪你赌**

### 5.5 两个恶魔的不同思路

#### 痢蛭 Lleech ⭐

**核心机制**：**首夜选宿主，宿主活着你不死**。

**宿主选谁**：
- **最难被处决的善方**——比如水手（永远不死）、茶艺师（藏身分高）
- **看起来无害的玩家**——不会被善方优先处决
- **不要选邪方队友**——浪费保护

**宿主作用**：
- 你被处决也不死 → 善方很难直接打死你
- 宿主中毒 → 他的信息全错，你可以利用他来误导善方
- 善方杀宿主时你才会死 → 别让他们意识到谁是宿主

**怎么用中毒的宿主造谣**：
- 宿主是博学者 → 他公布信息时**主动支持他**——其实是错的
- 宿主跳强神 → **让他的错信息流到关键决策**（处决某人的依据）

#### 小恶魔 Imp

**核心机制**：**标准恶魔，自杀传位**。

**正常刀人**：每晚选目标杀死（按 5.3 优先级）。

**传位时机**：
- 身份要暴露时（被善方锁定）→ **自杀传位**
- 选一个还活着的爪牙变成小恶魔
- 选谁？**威胁度低、看起来善方的爪牙**（如魔鬼代言人）

**传位策略**：
- 传位后旧小恶魔**死亡**，新小恶魔接班继续杀
- 善方以为"恶魔死了"开始庆祝——你给他们一记反杀
- **要算清楚**：传位后场上人数能不能让邪方继续打

## 六、拿到外来者怎么玩

### 落难少女 Damsel

**核心规则**：所有爪牙知道你在场。任何爪牙公开猜你 = 善方输。

**怎么活**：
- **绝对不跳身份**——一跳就送
- **装一个不矛盾的强神**——比如装博学者，让爪牙不敢猜你（怕猜错浪费机会）
- 关注谁在"试探"你——多次接近你想验证你是不是落难少女的爪牙
- 配合其他善方装"群落难少女"——让爪牙赌错

### 政客 Politician

**核心规则**：如果你是阵营落败的元凶，转邪获胜（即使已死）。

**怎么活**：
- **激进表演**：误导大家、提名错的人、推动错误处决
- **但别太离谱**——太明显被怀疑是邪方
- 关键：**说书人主观判断"最大责任"**——所以你的表现要让说书人觉得"是你害的"

### 理发师 Barber

**核心规则**：你死后，当晚恶魔可以换两人角色。

**怎么玩**：
- **死了反而有用**——你死后场上角色洗牌
- 但邪方知道这点，**可能故意不刀你**——你要主动让自己被处决吗？取决于场上形势
- 死后的"换角色"是恶魔决定，你没法控制——但能让"假身份"被掀开

### 莽夫 Goon

**核心规则**：**首个使用能力选中你的玩家**会醉，你转他的阵营。

**怎么玩**：
- 你可能从善变邪，也可能保持善
- **被信息系角色选中** → 你转善方阵营，你的善方信息变废（因为他醉了）
- **被邪方选中** → 你转邪方，要开始装坏人
- 关注自己的阵营变化——说书人会告诉你

### 异端分子 Heretic

**核心规则**：你在场，胜负对调（即使已死）。

**怎么玩**：
- **保持低调**——别让邪方一上来就刀你
- 邪方知道你存在的话会立刻刀你（不刀你，他们赢=输）
- 装一个无关紧要的角色——比如食人族（被动型）
- 死了也照样翻盘——所以**别因为怕死而急着跳**

## 七、跳身份指南

### 善方什么时候跳

| 时机 | 适用 |
|---|---|
| **D1 立即跳** | 信息系：占卜师、博学者、将军 — 越早跳信息越早流动 |
| **D2 跳** | 杂耍艺人（要先公开猜 5 次，D2 才能验证） |
| **被怀疑时跳** | 食人族、舞蛇人 — 看局势决定 |
| **永远别跳** | 茶艺师、巡山人 — 跳了能力废 |
| **关键决策前跳** | 渔夫 — 用能力时再亮 |

### 邪方该跳什么

回顾 5.2 节：**博学者 > 占卜师 > 将军 > 杂耍艺人**。

### 善方如何识破假身份

#### 信息对账法
- 两个跳"占卜师"的人 → 至少一个是假
- 占卜师 + 博学者 + 将军的信息互相矛盾 → 有人在编

#### 时间线还原法
- 邪方跳的身份必须**能力时机一致**——比如跳博学者就要每天问说书人
- 邪方跳"渔夫"但从来没用能力 → 可疑

#### 行为模式法
- 邪方倾向于**激进推动**（哥布林、政客、精神病患者）
- 善方信息系倾向于**先观察后跳**
- 一开局就疯狂指认别人的人 → 大概率邪方
